Instructions

You will update this React application to display a table of names that may be filtered by either first name or last name.

Requirements

  • there should be one component that controls the state and handles all events for the filter and results sections
  • the filter dropdown should allow you to select either first or last name
  • as you type in the filter text box, the results should update to display only the filtered names
  • the ./src/data/names.json file contains an array of first and last names that should be used with this example
  • changes should only be made within the src directory

Getting Started

  • ensure the project dependencies are installed 
    npm install
  • run the application 
    npm start
